## 下载和安装

### 方式一：DMG 安装程序（推荐）

1. 下载 `通知接收器.dmg` 文件
2. 双击打开 DMG 文件
3. 将应用程序拖动到 Applications 文件夹中
4. 在 Applications 文件夹中右键点击应用，选择“打开”
5. 如果遇到安全警告，点击“仍然打开”

### 方式二：压缩文件安装

1. 下载 `通知接收器.app.zip` 文件
2. 解压到任意位置
3. 右键点击解压后的应用，选择“打开”
4. 按照提示操作

## 使用说明

### 启动应用

1. 双击 Applications 文件夹中的应用程序
2. 应用会在 macOS 状态栏显示铃铛图标

### 接收通知

1. 应用会自动在后台启动 HTTP 服务器（端口 19999）
2. Android 设备发送通知到 `/notify` 接口
3. 应用会显示系统通知并保存历史记录

### 查看历史记录

1. 点击状态栏铃铛图标
2. 选择“查看历史记录”
3. 在弹出窗口中查看所有接收的通知

### 配置选项

- 支持自定义端口（修改 server.py 中 port 参数）
- 支持自定义通知图标（替换 notification-icon.icns）
- 支持过滤特定应用的通知（修改 notification_app.py）
